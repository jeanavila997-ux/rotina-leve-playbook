## Decisões importantes
- Tom: leve, sem culpa, sem diagnósticos clínicos, sem promessas médicas
- Estrutura fixa por capítulo: Cenário Real → Dor → Por que acontece? → Neurociência simples → Solução → Exercício → Resumo → Transição
- Compilação via `python scripts/compile.py`
- Exportação PDF via `python scripts/export_pdf.py`
- Servidor local: `python -m http.server 8000` → `http://localhost:8000/landing/`

## Comandos úteis para retomar
```powershell
# Ativar ambiente virtual
. .venv\Scripts\Activate.ps1

# Recompilar ebook
python scripts\compile.py

# Regenerar PDF
python scripts\export_pdf.py

# Testar landing local
python -m http.server 8000
```
