## Decisões importantes
- **PDF no assets**: O PDF foi movido para dentro de `landing/assets/` para que seja empacotado junto com o site estático na hospedagem.
- **Mockup CSS**: Mantido o mockup em CSS puro no hero devido à performance instantânea, com a imagem Open Graph servindo para compartilhamento social.
- **Build unificado**: A compilação e exportação continuam sendo feitas via `python scripts/compile.py` e `python scripts/export_pdf.py`.

## Comandos úteis para retomar
```powershell
# Ativar ambiente virtual
. .venv\Scripts\Activate.ps1

# Recompilar ebook
python scripts/compile.py

# Regenerar PDF em ambas as pastas
python scripts/export_pdf.py

# Testar landing localmente
python -m http.server 8000
```
