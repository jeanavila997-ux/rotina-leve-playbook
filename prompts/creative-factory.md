# Prompt — Fábrica de Criativos Rotina Leve

Crie 10 criativos para vender o produto **Rotina Leve — Playbook Anti-Procrastinação para Mentes Aceleradas**.

Para cada criativo, entregue:

- Título
- Hook
- Formato ideal
- Legenda curta
- CTA
- Prompt de imagem
- Variação segura para anúncio pago

Público:
- estudantes
- programadores
- criadores de conteúdo
- pessoas com dificuldade de começar tarefas

Tom:
- prático
- leve
- sem culpa
- moderno
- direto

Evite:
- diagnóstico
- promessa médica
- exageros
- antes/depois falso
